# FIFA World Cup 2026 Prediction Game ⚽🏆

A simple World Cup prediction game built with HTML, CSS, JavaScript, and Firebase.

## Play Here

https://mimimilkk.github.io/worldcup2026/

## How It Works

Predict the score for every FIFA World Cup 2026 match.

### Points System

| Result              | Points |
| ------------------- | ------ |
| Exact score         | 5      |
| Correct winner/draw | 3      |
| Wrong prediction    | 0      |

/with +25 points bonus if you win Champion Prediction/

/You can customize the scoring rules in the code./

### Rules

* Predictions are locked when the match starts.🔥
* Scores are automatically synced across all devices.
* Leaderboard updates automatically.

## Create Your Own Prediction Game

You can create your own version in less than 10 minutes.

### Step 1 — Copy Repository

Fork this repository or download the source code.

### Step 2 — Create Firebase Project

1. Go to Firebase Console
2. Create a new project
3. Enable Realtime Database
4. Copy your Firebase configuration

Replace the Firebase configuration in:

```javascript
const firebaseConfig = {
  ...
};
```

### Step 3 — Create GitHub Repository

Create a new GitHub repository.

Example:

```text
worldcup2026-office-pool
```

Upload all files from this project.

### Step 4 — Enable GitHub Pages

Settings → Pages

Source:

```text
Deploy from a branch
```

Branch:

```text
main
```

Folder:

```text
/ (root)
```

Your site will be available at:

```text
https://YOUR_USERNAME.github.io/REPOSITORY_NAME/
```

### Step 5 — Share With Friends

Send the link to your friends and start predicting.

No installation required.

Works on:

* iPhone
* Android
* Tablet
* Desktop
